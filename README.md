# Student-Registration-System-
A responsive Student Registration System built using HTML, CSS, and JavaScript that allows users to add, edit, and delete student records with real-time validation and persistent data storage using localStorage.



## Features
- Add student details
- Edit student records
- Delete records
- Data stored using localStorage
- Fully responsive design
- Input validation

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download project
2. Open index.html in browser
